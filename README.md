# own_gulp_setting

Reusable Gulp setup from early HTML/CSS/Sass practice work.

## What is included

- BrowserSync local server
- Sass compilation
- CSS minification
- Autoprefixer
- file watching for Sass and HTML changes

## Expected structure

```text
src/
  index.html
  sass/
  css/
```

## Run

```bash
npm install
npm start
```

## Status

Legacy tooling practice. This is useful as a small build setup reference, not as a standalone app.
